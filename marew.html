Cloning Practice using Desktop git app

The clone command in git is used when you want to download an existing git repository to your local computer. This means that you would have a clone or duplicate of the git repository and can start working on it whenever convenient.

steps to clone when using window terminal

  • 1. Create a new repo on github online
  • 2. Copy the httpy link and bring to on local desktop terminal
  • 3. write git then paste here the url
  • 4. then create a file .html
  • 5. git add . to add the file on the git
  • 6. commit/ to save using git commit
  • 7. git push to transfer on github online
